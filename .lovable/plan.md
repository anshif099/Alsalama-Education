
## Alsalama Education Campaign Landing Page

### Overview
A premium, mobile-first landing page for Alsalama Education with a blue + white medical-academic theme, featuring a hero section with lead capture form.

### Structure

1. **Header** — Alsalama logo centered at top with subtle background pattern (abstract medical/geometric shapes using CSS)

2. **Hero Section**
   - Bold headline: "Build Your Future in Healthcare with AI-Powered Courses"
   - Trust subheading with placement, certification, career focus
   - Trust badges row: "AI Integrated Courses", "Career Focused", "NABH Certified"

3. **Lead Capture Form Card** — Centered glassmorphism card with soft shadow
   - Full Name, Phone Number, Email Address, Location inputs
   - Course Selection dropdown (all 11 courses listed)
   - Comment/Message textarea
   - Large gradient blue "Apply Now" CTA button
   - Smooth hover/focus animations on all fields

4. **Footer** — Contact info, subtle branding

### Design
- Blue (#0066CC range) + white palette matching the logo
- Glassmorphism card with backdrop blur and soft shadows
- Rounded corners, clean spacing
- Subtle animated background shapes (CSS-only)
- Fully responsive mobile-first layout
- Logo image embedded from uploaded file

### Technical
- Single-page React component
- Form with client-side validation using state
- Toast notification on submit
- All styling via Tailwind CSS
- Logo copied to `src/assets/`
